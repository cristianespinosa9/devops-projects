# Fase 1.1 — Linux a fondo

Proyecto de fundamentos Linux orientado a DevOps. Se trabaja sobre la VM Ubuntu del lab k8s.

## Objetivo

Ganar fluidez en Linux para poder administrar servidores, debuggear pods, escribir scripts de automatización, y entender lo que hace Kubernetes bajo el capó.

## Entorno

- VM Ubuntu 22.04 en Hyper-V (192.168.1.193)
- Acceso SSH: `ssh k8s-lab`
- Usuario: `cristian`
